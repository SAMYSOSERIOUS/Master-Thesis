# docs

Documentation and generated reports for the thesis pipeline (no executable notebooks).

This folder holds written outputs — progress reports, result summaries and
generated metadata — that accompany the experiments in the sibling folders.
Unlike the other folders, it contains no `.ipynb` files and therefore no methods
of its own; it is the place where results produced by `pipeline_v1` and the
analysis notebooks are recorded in human-readable form.

## Contents

- `koa_results.docx` — written summary of knee-osteoarthritis grading results.
