# notebooks_standardized

Top-level collection of all standardized thesis notebooks and supporting assets.

This directory was standardized to snake_case naming for folders and files.

## Subfolders
- `calibration`: Calibration and stability experiments for model tuning and ensemble behavior.
- `diagnostics`: Cross-dataset diagnostic analyses and comparison notebooks.
- `docs`: Documentation folder for reports, notes, and generated metadata files.
- `experiments`: Additional and bonus experimental notebooks, ablations, and helper studies.
- `mendeley`: Mendeley dataset acquisition and preparation notebooks.
- `mrkr_2`: Contains related notebooks and resources.
- `mt3_2`: Contains related notebooks and resources.
- `nhanes3`: NHANES3 ingestion and label-processing notebooks.
- `oai`: OAI dataset download, preprocessing, and label pipeline notebooks.
- `oarsi_2`: Contains related notebooks and resources.
- `pipeline_v1`: Primary end-to-end training and evaluation pipeline sequence (00-14).

## Files
- `dataset_status.ipynb`: Notebook: dataset status.
